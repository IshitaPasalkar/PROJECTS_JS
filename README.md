# JavaScript Projects

This repository contains a collection of beginner-friendly JavaScript projects created while learning web development.

## Projects Included

### 1. Age Eligibility Checker

Checks whether a user is eligible based on age input.

### 2. Auto Typing

Creates a typing animation effect using JavaScript.

### 3. Countdown Timer

Displays a countdown timer that updates in real time.

### 4. Counter App

Increase, decrease, and reset a counter using button controls.

### 5. Dynamic JavaScript Project

Practice project demonstrating JavaScript DOM manipulation.

### 6. Know Your Animal Sound

Interactive project that plays or displays animal sounds.

### 7. Loan Calculator

Calculates loan payments based on user input.

### 8. Number Adder

Adds numbers entered by the user and displays the result.

### 9. Sound Board

Plays different sound effects using buttons.

### 10. Temperature Converter

Converts temperatures between Celsius and Fahrenheit.

### 11. Guess Number Game

A simple number guessing game built with JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Purpose

These projects were built to practice:

* DOM Manipulation
* Event Handling
* Functions
* Variables and Data Types
* User Input Processing
* Interactive UI Development
